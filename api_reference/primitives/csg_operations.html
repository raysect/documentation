2.3. CSG Operations¶

class raysect.primitive.csg.CSGPrimitive¶

Constructive Solid Geometry (CSG) Primitive base class.

This is an abstract base class and can not be used directly.

class raysect.primitive.csg.CSGRoot¶

Specialised scenegraph root node for CSG primitives.

The root node responds to geometry change notifications and propagates them to the CSG primitive and its enclosing scenegraph.

class raysect.primitive.csg.NullPrimitive¶

Dummy primitive class.

The _CSGPrimitive base class requires a primitive that returns a valid bounding box. This class overrides the bounding_box method to return an empty bounding box. This class is intended to act as a place holder until a user sets a valid primitive.
